Para ejecutar la clase SmartWalletTest, debes usar Maven desde la terminal:

Ubícate en la raíz del proyecto (donde está el archivo pom.xml).
Ejecuta el siguiente comando:
mvn test

Maven automáticamente:
  Compila el código fuente (src/main/java)
  Compila los tests (src/test/java)
  Ejecuta todos los métodos anotados con @Test
